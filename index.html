iTalenta Inc is the leading provider of professional chemistry and bilogy software. We offer a suite of very professional cross-platform software tools that provide reusable libraries and end-user applications for computational chemistry, biology science, and related areas.


Avogadro 2 is a versatile desktop application for editing and visualizing molecular data. It is a cross-platform application that can easily be extended with C++ plugins, or Python scripts. Support for a number of computational chemistry codes is offered.

MoleQueue provides desktop integration of high-performance computing (HPC) resources, along with local execution of standalone code. It is a small, Qt-based, system-tray resident application that makes computing resources easier to access from graphical applications.

MongoChem uses MongoDB to store chemical data, and reuses Avogadro 2, VTK, and elements of Open Babel to provide desktop cheminformatics. The application provides a view of large collections of molecules, enabling you to find the ones with the properties you are most interested in.
